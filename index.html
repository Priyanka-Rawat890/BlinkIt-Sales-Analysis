BlinkIt E-commerce Sales Analysis

End-to-end project analyzing sales performance using Excel, SQL, Python & Power BI

Project Overview

This project provides detailed insights into BlinkIt’s sales data focusing on sales trends, customer satisfaction, and inventory across outlets using KPIs and dashboards.

Tools & Technologies

  • Excel - Data preparation, pivot tables, dashboard creation
  • SQL - KPIs & data extraction queries
  • Python (Pandas, Matplotlib, Seaborn) - Data analysis & visualizations
  • Power BI - Interactive dashboards with filters & KPIs

Business Requirements

  • Analyze sales across outlets and product categories
  • Measure customer satisfaction scores
  • Evaluate inventory distribution effectiveness
  • Create dashboards to support decision-making

Key KPIs

  • Total Sales - Overall revenue generated
  • Average Sales - Revenue per transaction
  • Number of Items Sold
  • Average Rating - Customer satisfaction levels

Project Workflow

  1. Cleaned and processed data in Excel
  2. Created Excel dashboards with pivot tables
  3. Wrote SQL scripts for data retrieval and KPIs
  4. Analyzed data and created visuals using Python
  5. Developed interactive Power BI dashboards

Project Files & Links

Key Insights

  • Low Fat items contributed more to total sales than Regular ones
  • Snack foods and dairy are the highest-selling categories
  • Supermarket Type 1 outlets generate the most revenue
  • Tier 3 cities outperform Tier 1 and 2 in sales
  • Outlet size & establishment year influence sales distribution
